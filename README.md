# PyQGIS Cartography Scripts

A collection of PyQGIS scripts for professional cartographic workflows in QGIS. Each script is designed to be pasted into the QGIS Python Console and run interactively.

## Scripts

- [Slope-Aligned Contour Label Generator](scripts/contour_labeler/) — places elevation labels along user-drawn lines, rotated perpendicular to slope direction.
- [Map Sheets Along Route](scripts/map_sheets_along_route/) — generates oriented, numbered map sheet polygons along a route for use as an Atlas coverage layer.

## Requirements

- QGIS with Python Console access — no plugin installation needed.
- A projected CRS in metres is recommended for accurate geometry.
